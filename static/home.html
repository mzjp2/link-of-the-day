Link of the Day

Opening qb.zainp.com/link will forward you to the link of the day.


To contribute a LotD, send a POST or PUT request to qb.zainp.com/link with Content-Type header set to application/x-www-form-urlencoded, e.g:


                    $ curl -H "Content-Type: application/x-www-form-urlencoded" -d "url=https://quantumblack.com" qb.zainp.com/link -X POST
